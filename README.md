# Lord of Insomnia

A sad-themed personal writing space hosted on GitHub Pages.
Words written at 3 a.m. — a place to write and be read.

## View the site

> **[orgito1015.github.io/My-twitter-Lord-of-Insomnia](https://orgito1015.github.io/My-twitter-Lord-of-Insomnia)**

## Adding a new post

1. Copy `posts/the-3am-feeling.html` as a starting template
2. Edit the title, date, and body text inside the file
3. Add a new `<a class="post-card" …>` entry in `index.html`
4. Commit and push — GitHub Pages will publish it automatically

## Theme

- Deep midnight background with animated rain
- Lora serif + Inter sans-serif typography
- Muted blues and grays — melancholic but readable
